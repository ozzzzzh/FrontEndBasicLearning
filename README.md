Frontend Basic Learning
=======================

简介
----

这是一个用于系统学习前端基础的练习仓库，配套学习路线见 `learningMap.md`。内容主要围绕：

- HTML：从浏览器渲染流程到语义化结构与可访问性
- CSS：盒模型、布局系统、视觉风格与动画
- JavaScript：语言基础、DOM 操作、事件与异步
- 前端工程化：模块化、构建工具与依赖管理
- Vue 等框架抽象与前后端协作

目录结构（示意）
---------------

- `learningMap.md`：整体学习路线与阶段目标
- `Chapter1/`：HTML 与浏览器相关练习
- 后续章节：CSS、JavaScript、工程化和框架等内容

使用方式
--------

- 按 `learningMap.md` 推荐顺序依次学习，不要跳跃
- 每个小节配合代码示例与练习文件动手操作
- 建议在浏览器 DevTools 中多做实验、观察渲染与无障碍结构

学习建议
--------

- 不急于求成，每个知识点尽量“能讲出来、能写出来、能 debug”
- 多用注释与小实验文件验证自己的理解
- 定期回顾 `learningMap.md` 中的“掌握标准”，对照自查
